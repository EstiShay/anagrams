## Specifications

| Behavior      | Example Input         | Example Output        |
| ------------- | ------------- | ------------- |
| User only gives 1 word | Dog | False |
| User gives two words that are not anagrams | Dog Cat | False |
| User gives two words that are anagrams | Cat Act | True |
